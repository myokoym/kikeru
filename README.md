# Kikeru - Keyboard Friendly Audio Player

A keyboard friendly audio player by Ruby/GStreamer.

## Installation

    $ gem install kikeru

## Usage

### Launch

    $ kikeru [OPTION]... [FILE]...

If no argument, then search current directory.

### Options

    -d, --deep
        deep search as "**/*"

### Keybind

    n: next
    p: prev
    r: restart
    q: quit
    space: pause/play
